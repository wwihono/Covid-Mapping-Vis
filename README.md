# Covid-19 in the United States(2020) - Chloropleth and Proportional Symbols Map

## Project Overview
This project visualizes the spatial distribution, of COVID-19 cases in the United States in 2020
The visualisation uses two techniques:
1. Map 1: Chloropleth showing covid case rate per 1000 people by county
2. Map 2: Proportional symbols map showing total case counts by county
